## Персональный портфолио-сайт + блог на MDX

### ТЗ проекта

Портфолио-сайт с блогом на MDX для фулстек-разработчика. Статьи как .mdx файлы.

**Ключевые фичи:**

- 6 страниц: главная, проекты, блог, статья, обо мне, контакты
- Блог на MDX: frontmatter, подсветка синтаксиса, оглавление, навигация между статьями
- Тёмная/светлая тема (next-themes)
- RSS-лента, sitemap.xml, JSON-LD

**Стек:** Next.js 14+, TypeScript, Tailwind, ShadCN, next-mdx-remote, rehype-pretty-code, next-themes, Railway

---

## Промпты

### 1. Инициализация ShadCN + компоненты

Инициализирует ShadCN UI и компоненты для портфолио.

```
Инициализируй ShadCN UI с темой "default". Установи компоненты: button, 
card, badge, separator, tabs, avatar, navigation-menu.
```

### 2. Генерация портфолио + блог (основной)

Генерирует весь портфолио-сайт с блогом.

```
Создай персональный портфолио-сайт с блогом на MDX на основе CLAUDE.md.

Порядок:
1. Зависимости: next-mdx-remote, gray-matter, rehype-pretty-code, remark-gfm
2. Утилиты для MDX (чтение файлов, парсинг frontmatter, slug)
3. Layout: навигация + переключатель темы, футер
4. Главная: Hero + Featured проекты + последние статьи
5. Проекты с фильтрацией по стеку
6. About: навыки с прогресс-барами + таймлайн
7. Блог: список статей с поиском и фильтром по тегам
8. Страница статьи: MDX, оглавление, подсветка кода, навигация
9. RSS feed (/feed.xml)
10. Sitemap и robots.txt
```

### 3. Создание статей блога

Создаёт 3 демо-статьи в MDX.

```
Создай 3 примера статей блога в MDX:
1. "Как я стал разработчиком за 6 месяцев"
2. "Топ-5 инструментов для вайб-кодинга в 2025"
3. "Почему Next.js — лучший фреймворк для портфолио"

С frontmatter: title, description, date, tags, image, published.
```

### 4. Оглавление (Table of Contents)

Интерактивное оглавление с подсветкой текущего раздела.

```
Создай компонент TableOfContents для статьи:
1. Автоматическое оглавление из h2 и h3
2. Sticky на десктопе
3. Подсветка текущего раздела (Intersection Observer)
4. На мобильных — аккордеон вверху
```

### 5. RSS и SEO

RSS-лента, sitemap, JSON-LD.

```
Добавь:
1. RSS-ленту /feed.xml
2. Динамический sitemap.xml
3. JSON-LD: Person на главной, Article на статьях
4. Мета-тег link rel="alternate" для RSS в layout
```

### 6. Переключатель темы

Dark/light тема с сохранением.

```
Реализуй переключение темы:
1. next-themes
2. Кнопка Sun/Moon в хедере
3. Сохранение в localStorage
4. По умолчанию — системная тема
5. Плавная анимация переключения
```
