## Overview

A versatile Telegram bot powered by GPT-4 that serves as a personal AI assistant. Features scheduling, reminders, document summarization, and general Q&A.

## Features

- **Smart Q&A** — answers questions with GPT-4, cites sources when available
- **Document summarization** — send a PDF or web URL, get a summary
- **Scheduling** — set reminders and recurring tasks via natural language
- **Conversation history** — remembers context within a session
- **Admin panel** — usage statistics and user management

## Technical Stack

- **python-telegram-bot** v20 with async support
- **OpenAI GPT-4** with function calling
- **PostgreSQL** for user data and reminder storage
- **APScheduler** for reminder delivery
- **Docker** + **Railway** for deployment

## Results

- 500+ active users
- 10,000+ messages processed monthly
- Available 24/7 with 99.9% uptime
