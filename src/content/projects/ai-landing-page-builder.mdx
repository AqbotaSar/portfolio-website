## Overview

A web app where you describe your product in plain text and get a complete, conversion-optimized landing page. GPT-4 generates the copy and structure; the app outputs clean Next.js + Tailwind code.

## Features

- **One-input generation** — describe product → full landing page
- **Multiple templates** — SaaS, e-commerce, services, portfolios
- **Live preview** — see the result before downloading
- **Export options** — download as Next.js project or static HTML
- **SEO optimization** — generated pages include meta tags, OG images

## Results

- 200+ landing pages generated
- Average generation time: 15 seconds
