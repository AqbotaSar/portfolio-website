## Overview

A universal web scraper that uses GPT-4 vision to understand page layouts and extract data without custom CSS selectors. Describe what you want — the AI figures out how to get it.

## Features

- **Zero-config extraction** — describe data in plain English, AI extracts it
- **JavaScript rendering** — handles SPAs and dynamic content via Playwright
- **Anti-detection** — rotating proxies, human-like behavior simulation
- **Scheduled scraping** — run jobs on cron schedule with change detection
- **REST API** — integrate scraping into any workflow via API

## Results

- Replaced 8 separate site-specific scrapers with one tool
- 90% reduction in scraper maintenance time
